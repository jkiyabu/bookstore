Bookstore and Reviews
---------------------
A site made with Drupal where a user can add book reviews for any one to visit and read.


Installation
------------
* Clone repository
* Set home directory as document root in MAMP web server
* Import dbase bookstore.sql.zip from "sites/all/db-backup" using php myadmin
* Create "bookstore" user for database with password also set to "bookstore"
* Browse to localhost:8888
* Log in as username "bookstore" and password "bookstore"
